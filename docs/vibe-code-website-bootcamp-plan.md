# Vibe Code Website Bootcamp

เอกสารแผนคอร์สและ positioning สำหรับปรับหน้าเว็บไซต์ Best Solutions Skill

## Positioning

**Vibe Code Website Bootcamp** คือคอร์สสอนสด 2 วัน สำหรับคนที่อยากใช้ AI สร้างเว็บไซต์จริง พร้อมติดตั้งเครื่องมือวัดผลพื้นฐาน เข้าใจการ Deploy และรู้วิธีต่อยอดเว็บเป็น Portfolio สมัครงาน รับงาน หรือสร้างรายได้

คำสำคัญที่ใช้แทนคำเดิม:

- ใช้คำว่า **Vibe Code** แทน **Custom Code**
- สื่อว่าไม่ต้องเริ่มจากการเขียนโค้ดลึกทั้งหมด แต่ต้องเข้าใจ Workflow การใช้ AI ทำเว็บจริง
- เน้นผลลัพธ์เป็นเว็บที่ใช้งานได้จริง วัดผลได้ และต่อยอดด้านอาชีพได้

## กลุ่มเป้าหมาย

1. **คนทำงานที่อยากอัปสกิล AI + Website**
   - Marketer, Content Creator, Admin, AE, Designer, Project Coordinator, Business Development
   - อยากทำเว็บเองได้ หรือคุยกับทีม Dev/Designer รู้เรื่องขึ้น

2. **เจ้าของธุรกิจ / SME / ฟรีแลนซ์**
   - อยากทำ Landing Page, เว็บบริการ, เว็บรับ Lead, เว็บ Portfolio หรือเว็บธุรกิจเอง
   - ต้องการลดการพึ่งพาทีมเทคนิคในงานพื้นฐาน

3. **นักศึกษา / คนเริ่มต้นสายงานดิจิทัล**
   - อยากมี Portfolio เป็นเว็บไซต์จริง
   - ใช้ต่อยอดสมัครงานสาย Web, Digital Marketing, UX/UI, Content, AI Assistant หรือ Junior Digital Role

4. **คนที่อยากรับงานเสริมหรือสร้างรายได้**
   - อยากทำเว็บ Landing Page, เว็บบริษัทขนาดเล็ก, เว็บโปรไฟล์ หรือเว็บรับฟอร์มติดต่อ
   - ต้องการ Workflow ที่นำไปทำซ้ำกับลูกค้าหรือโปรเจกต์ตัวเองได้

## จุดขายหลัก

- เรียนสด 2 วัน รวม 10 ชั่วโมง
- สร้างเว็บไซต์จริงด้วย Vibe Code และ AI
- ไม่จำเป็นต้องเริ่มจากพื้นฐานเขียนโค้ดลึก
- ได้เว็บที่ Deploy ออนไลน์ได้
- ได้เรียนการติดตั้งเครื่องมือวัดผลเว็บไซต์พื้นฐาน
- มี Consult หลังเรียนจบ
- ถามต่อได้ใน Community
- มี Replay, Prompt Pack, Starter Template และ Website Launch Checklist
- เน้นต่อยอดอาชีพ: ทำ Portfolio, สมัครงาน, รับงาน, ทำเว็บธุรกิจ หรือสร้างรายได้เสริม
- สอนโดยทีม Agency ที่ทำเว็บไซต์และการตลาดออนไลน์จริง

## โครงสร้างคอร์ส

### Day 1: วางแผน ออกแบบ และสร้างเว็บด้วย AI

#### 10:00-12:00: Module 1 - Vibe Code Foundation

- Vibe Code คืออะไร
- ใช้ AI ช่วยทำเว็บได้อย่างไร
- Workflow การทำเว็บไซต์ยุค AI
- เลือกประเภทเว็บที่จะทำ: Landing Page, Portfolio, เว็บธุรกิจ, เว็บรับ Lead
- วางเป้าหมายเว็บ กลุ่มเป้าหมาย จุดขาย และ CTA
- เขียน Project Brief ให้ AI เข้าใจ

#### 13:00-16:00: Module 2 - Website Design & HTML Preview

- วาง Sitemap / Section / User Flow
- UX/UI พื้นฐานที่จำเป็น
- Layout, Typography, Color, Spacing, Responsive
- ใช้ AI ช่วยสร้าง Wireframe และหน้าเว็บ
- ทำ HTML Preview
- ปรับหน้าเว็บให้ดูน่าเชื่อถือ พร้อมต่อยอดเป็นเว็บจริง

### Day 2: ทำเว็บจริง วัดผล Deploy และต่อยอดรายได้

#### 10:00-12:00: Module 3 - Build Website with Next.js

- เข้าใจโครงสร้าง Next.js เบื้องต้น
- แปลง HTML Preview เป็น Component
- แก้ข้อความ รูป สี และ Section
- ใช้ AI ช่วย Debug Error
- เตรียมเว็บให้พร้อม Deploy

#### 13:00-16:00: Module 4 - Tracking, Deploy & Career Roadmap

- ทำ Contact Form / Lead Form เบื้องต้น
- แนวคิดการเชื่อม Supabase
- Deploy เว็บไซต์ขึ้น Vercel
- ติดตั้ง Google Analytics 4 พื้นฐาน
- เข้าใจ Google Tag / Google Tag Manager เบื้องต้น
- รู้จัก Google Search Console สำหรับดู SEO และ Indexing
- รู้จัก PageSpeed Insights สำหรับตรวจ Performance
- แนวคิด Conversion / Key Event เช่น กดปุ่ม LINE, ส่งฟอร์ม, คลิก CTA
- Checklist ก่อนส่งเว็บจริง
- วิธีใช้เว็บเป็น Portfolio
- แนวทางสมัครงาน รับงาน และตั้งต้นสร้างรายได้

## Google Tools ที่สอนในคอร์ส

- **Google Analytics 4 (GA4):** ดู Traffic, Users, Events และพฤติกรรมผู้ใช้
- **Google Tag / Google Tag Manager:** ติดตั้ง Tracking และจัดการ Tag
- **Google Search Console:** ดู Performance บน Google Search, Indexing และ Keyword เบื้องต้น
- **PageSpeed Insights:** ตรวจความเร็วและ Core Web Vitals
- **Google Ads Conversion Tracking:** แนะนำระดับพื้นฐานสำหรับคนที่ต้องการต่อยอดยิงโฆษณา

## AI Tools Highlight

ส่วนนี้ใช้เป็นแรงบันดาลใจสำหรับ Section "Tools Update" บนหน้าเว็บ โดยสื่อว่าเครื่องมือไม่ได้ถูกสอนแยกเป็นเครื่องมือทีละตัว แต่แทรกอยู่ใน Workflow จริงตามความเหมาะสมของงาน

- **Claude:** วางแผน ช่วยคิดโครง สร้าง Prompt และช่วยเขียนโค้ด
- **ChatGPT / Codex:** คิด Prompt, Refactor, Debug และช่วยอ่าน Error
- **Gemini:** ช่วยคิดและตรวจงานเพิ่มเติม
- **Google Stitch:** ช่วยสำรวจไอเดีย UI และแนวทางหน้าตาเว็บไซต์
- **Figma Make:** ต่อเชื่อมงาน Design และ Prototype
- **Figma MCP:** เชื่อม Design Workflow กับการทำงานด้วย AI
- **design.md:** บันทึก Design System, Token, Guideline และแนวทาง UI
- **skill.md:** สร้าง Skill / Prompt Workflow สำหรับทำงานซ้ำได้เป็นระบบ
